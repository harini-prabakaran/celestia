## ☆ Celestia ☆

Celestia is a machine learning project that explores exoplanets using data from the NASA Exoplanet Archive.

### Objectives

* Discover planetary archetypes using unsupervised learning.
* Explain the factors that define those archetypes.
* Investigate whether host-star properties influence planetary populations.

### Methods

* Data Cleaning & Feature Engineering
* K-Means Clustering
* Classification
* SHAP Explainability
* Statistical Analysis
* Streamlit Dashboard

### Key Findings

* Identified four distinct planetary archetypes.
* Planetary mass and density were the strongest drivers of archetype assignment.
* Host-star mass, radius, temperature, and luminosity differed significantly across archetypes.
* Host-star characteristics refined planetary archetypes by revealing additional stellar-environment patterns.

### Dataset

NASA Exoplanet Archive (PSCompPars) 

### Technologies

Python, Pandas, NumPy, Scikit-Learn, SHAP, Matplotlib, Seaborn, Streamlit
