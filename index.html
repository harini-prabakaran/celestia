✦ Final Project Conclusions
Machine Learning Can Effectively Discover Meaningful Exoplanet Population Patterns
This project developed an end-to-end machine learning pipeline using data from the NASA Exoplanet Archive to explore whether exoplanets can be grouped into meaningful population archetypes based on their physical characteristics.

Through feature engineering, logarithmic transformation, standardization, and K-Means clustering, four distinct exoplanet archetypes were identified: Small Dense Planets, Intermediate Low-Density Planets, Low-Density Giants, and Massive Dense Giants. Elbow Method and Silhouette Analysis both supported an optimal clustering solution of K = 4.

A supervised classification model was then trained to predict cluster membership from planetary features. SHAP explainability analysis revealed that planet mass and planetary density were the most influential variables driving archetype assignment, while orbital distance contributed additional but comparatively smaller predictive value.

Statistical analysis of host-star properties using the Kruskal–Wallis test demonstrated significant differences across the discovered planetary populations. Incorporating stellar features into a second clustering experiment showed that host-star characteristics can influence cluster structure, although the primary separation remains driven by planetary properties.

Overall, this project demonstrates the complete machine learning workflow of unsupervised learning, supervised learning, explainable AI, statistical hypothesis testing, and data visualization while providing interpretable insights into exoplanet population structure. The resulting framework can serve as a foundation for future work involving habitability analysis, anomaly detection, or more advanced astrophysical classification systems.
